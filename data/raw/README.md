# Local analysis data

Place the P1v2 analysis workbook here as:

`P1数据.xlsx`

The legacy filename `prediction_error_memory_calculation_ready.xlsx` is also
recognized when `P1数据.xlsx` is absent. An explicit path can be supplied with
the `PEM_WORKBOOK` environment variable or as the first command-line argument.

This directory is ignored by Git. The public repository must not contain copyrighted PDFs or a workbook that has not been cleared for public release.
