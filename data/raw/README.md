# Local analysis data

Place the P1v2 analysis workbook here as:

`P1v2数据.xlsx`

The strict `runP1v2.1` analysis checks this filename as well as the frozen
14 logOR + 5 SMD + 3 nonlinear row counts. An explicit path can be supplied
with the `PEM_WORKBOOK` environment variable or as the first command-line
argument, but its basename must still be `P1v2数据.xlsx` in strict mode.

This directory is ignored by Git. The public repository must not contain copyrighted PDFs or a workbook that has not been cleared for public release.
