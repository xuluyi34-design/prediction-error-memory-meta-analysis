# Generated results

Timestamped `runP1v2.1_YYYYMMDD_HHMMSS` analysis outputs are written here.
Generated files are ignored by Git so that confirmatory results are not
committed accidentally before the intended analysis and release stage.

`run_metadata.csv` confirms the data freeze, method/output version, workbook
filename, and expected 14 + 5 + 3 counts. `models/model_summary.csv` records the
selected model and inference method for each block, while
`models/descriptive_effects.csv` contains the individual effects for blocks
that were not pooled.
